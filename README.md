## 🚀 **Installation & Usage**

### **Option 1: Download Release (Recommended)**
```bash
# Download latest release
wget https://github.com/Thatkidtk/it-drive/releases/latest/download/it-drive-latest.zip

# Extract and run
unzip it-drive-latest.zip
cd it-drive
./it-tool/Start-macOS-Python.command
```

### **Option 2: Clone Repository**
```bash
git clone https://github.com/Thatkidtk/it-drive.git
cd it-drive
./it-tool/Start-macOS-Python.command
```

### **Usage Examples:**
```bash
# Full diagnostics with HTML report
python3 it-tool/mac/pytool.py audit --html --open

# Apply quick network fixes
python3 it-tool/mac/pytool.py fix --auto

# Interactive remediation menu
python3 it-tool/mac/pytool.py fix
```
